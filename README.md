# Shadows in the Grass 🌿🌑

Um jogo de sobrevivência e puzzle minimalista, onde a paciência e a observação são suas únicas armas contra o que espreita na grama alta.

## 🎮 Como Jogar
- **Objetivo**: Atravesse o labirinto e alcance o portal espiral no canto superior direito.
- **Movimentação**: Use as teclas `WASD` ou as `Setas` do teclado.
- **Sobrevivência**:
  - Grama Estática: Segura.
  - Grama Tremendo/Olhos Vermelhos: **Perigo!** Saia imediatamente ou não entre.
  - As sombras atacam em ciclos. Observe o ritmo antes de avançar.

## 🛠️ Tecnologias
- **HTML5 Canvas**: Motor de renderização.
- **Vanilla JavaScript**: Lógica de jogo e ciclos de estado.
- **CSS3**: Estética "dark mode" com efeitos de vinheta e transições suaves.
- **Google Fonts**: Tipografia 'Outfit' para uma presença moderna.

## 🚀 Como Publicar (GitHub Pages)
Este projeto foi preparado para ser hospedado no GitHub Pages:
1. No **GitHub Desktop**, adicione a pasta do projeto.
2. Faça o 'Commit' das alterações.
3. Clique em 'Publish Repository' (ou 'Push' se já estiver online).
4. No GitHub (site), vá em **Settings** > **Pages** e selecione a branch `main` como fonte.
5. Seu jogo estará disponível em `https://seu-usuario.github.io/labirinto/`.
